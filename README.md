    });
</script>

# ✅ 快速开始

```
1、git clone 或者 直接下载 本项目

2、cd 到项目根目录

3、yarn	//安装依赖

4、yarn pre.dev	// 预览 下的调试模式

5、yarn dev // 自己项目 下的调试模式

6、yarn pre.build	//预览 下的编译打包

7、yarn build	//自己项目 下的编译打包
```
